# Arete 更新日志

## v0.4 — 2026-04-08

### 图片系统
- 支持 Ctrl+V 粘贴 / 拖放图片到选中节点
- 节点内显示缩略图（160x120），双击打开全屏灯箱查看原图
- Markdown 输出包含 `![](url)` 图片语法
- 本地同步时 base64 图片自动提取为 `images/` 文件夹中的真实文件

### Markdown 输出优化
- 第一层子节点输出为 `##` 标题，深层用 4 空格缩进嵌套列表
- GitHub / Obsidian 均可正确渲染层级

### 品牌 & UI
- 新 favicon：斜体衬线 α + 品牌色圆角方块
- 页面 logo 与 favicon 统一风格
- 修复画布元素蓝色聚焦边框

### Tab 键 Bug 彻底修复
- 根因：SVG 元素被 Tab 聚焦后浏览器跳转到 data URL
- 修复：SVG / canvas 加 `tabindex="-1"` + `focusable="false"`
- 额外加 window capture 阶段全局拦截

## v0.1 — 2026-03-25 ~ 2026-03-31

### 核心功能
- 思维导图画布：拖拽、缩放、多根节点

### 布局 & 交互
- 紧凑自动布局算法，编辑后自动重排

### 数据同步
- 双向 Markdown 同步面板

- GitHub API 同步（推送 + 启动时拉取）

- 按钮触摸目标优化（36px / 40px）

- 色盲友好的提示点形状
