设置

GitHub 同步

如何设置？
  1. 打开 GitHub Token 页面，点击 Generate new token (classic)
  2. 勾选 repo 权限，生成后复制 Token 填入下方
  3. 填写你的 GitHub 用户名、仓库名（需提前创建好）
  4. 文件路径填写保存位置，如 notes/mindmap.md
  5. 勾选自动同步，之后每次编辑都会自动推送到 GitHub

需要 repo 权限。Token 仅存储在本地浏览器，不会上传到任何服务器。

本地文件夹同步

选择本地文件夹（如 Obsidian vault），每次保存时自动同步 Markdown 文件。

Key 仅存储在本地 localStorage，不会发送到除 Anthropic API 之外的任何地方。

✦ 启用 AI 协作

⌘G 可以让 AI 自动为选中节点生成子节点，帮你拓展思路。
输入 Anthropic API Key 即可启用。

↗ 前往 console.anthropic.com/keys 获取 Key
Key 仅存储在本地 localStorage，不会发送到 Anthropic API 以外的任何地方。
